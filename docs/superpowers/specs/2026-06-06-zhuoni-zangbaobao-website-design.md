# 卓尼县藏宝宝网络有限责任公司官网设计文档

## 概述

为卓尼县藏宝宝网络有限责任公司打造品牌展示型静态官网，采用全屏滚动叙事设计，突出"藏王故里·秘境卓尼"品牌形象。网站托管于 Cloudflare Pages。

## 公司背景

- **公司名称**：卓尼县藏宝宝网络有限责任公司
- **法定代表人**：俞卓玛
- **所在地**：甘肃省甘南藏族自治州卓尼县
- **定位**：卓尼县电商进农村综合示范项目承办企业，运营县域公共品牌"秘境卓尼"
- **主营业务**：高原绿色生态土特产的标准化、品牌化和电商化（食用菌、中藏药材、高原畜产、洮砚工艺品等8大品类）
- **文化名片**：藏王故里·秘境卓尼 | 中国洮砚之乡 | 连续五年中国最美县域

## 技术栈

| 层级 | 技术选型 | 理由 |
|------|---------|------|
| 框架 | Astro 5.x | 零JS默认输出、岛屿架构、SEO友好、天然适合静态展示站 |
| 样式 | Tailwind CSS 4.x | 实用优先、快速构建自定义设计系统 |
| 交互 | Alpine.js | 轻量级（~15KB），处理分类筛选、移动菜单等简单交互 |
| 部署 | Cloudflare Pages | 免费、无限带宽、全球CDN、自动HTTPS、Git自动部署 |
| 图片优化 | Astro Image + Cloudflare Image Resizing | 自动 WebP/AVIF 转换、响应式图片 |

## 设计风格

**现代藏风** — 现代简约为主，藏族文化元素点缀。

### 配色方案

| 角色 | 色名 | Hex | 用途 |
|------|------|-----|------|
| 主背景 | 高原米白 | #FAF0E6 | 大面积页面背景 |
| 卡片背景 | 雪白 | #FFFFFF | 内容卡片、表单背景 |
| 次要背景 | 麻布色 | #E8E0D0 | 次要区域、分隔区域 |
| 主强调色 | 藏红/绛红 | #860C1E | 标题、导航高亮、分隔线、CTA |
| CTA/装饰 | 古铜金 | #B6A766 | 按钮、装饰线条、图标 |
| 正文色 | 墨蓝 | #1A1A2E | 正文文字、深色区域 |
| 生态标签 | 草原绿 | #2D5016 | 有机/绿色认证标签 |
| 辅助色 | 大地棕 | #6B4C3B | 次要文字、边框 |

### 设计原则

1. **大面积留白** + 米白底色营造高端质感
2. **藏红点缀**：导航高亮、标题下划线、分隔线
3. **古铜金CTA**：按钮、行动召唤
4. **藏族纹样**：仅在页面顶部装饰线和 Footer 区域出现
5. **墨蓝文字**：正文使用深色墨蓝而非纯黑，更柔和高级

## 页面结构（共6个页面）

### 页面1：首页（Home）— 全屏滚动叙事

6个全屏板块，自上而下滚动浏览：

**板块1 - Hero 区域**
- 全屏大图背景（卓尼高原风光）
- 透明导航栏（Logo左对齐 + 菜单项）
- 品牌标语："藏王故里·秘境卓尼" + "来自雪域高原的馈赠"
- 副标题：公司定位描述
- CTA按钮："探索产品"

**板块2 - 关于我们（精选）**
- 左侧：公司简介摘要 + "了解更多"按钮
- 右侧：4个数据卡片（2×2网格）
  - 10万+ 服务人口
  - 136种 野生药材
  - 8大 产品品类
  - 5139 km² 服务区域

**板块3 - 产品展示（精选）**
- 标题："高原珍品·绿色甄选"
- 4个分类卡片网格（1行4列）：
  - 食用菌类（黑木耳·羊肚菌·姬松茸）
  - 中藏药材（当归·柴胡·黄芪）
  - 高原畜产（牦牛肉·藏羊肉·蕨麻猪）
  - 手工艺品（洮砚·刺绣·木雕）
- "查看全部产品"按钮

**板块4 - 文化故事（精选）**
- 深色背景（#1A1A2E）
- 标题："藏王故里·秘境卓尼"
- 5大文化横向卡片（1行5列）：
  - 土司文化、佛教文化、洮砚文化、觉乃民俗、自然生态

**板块5 - 新闻摘要**
- 3条新闻列表，左侧日期（古铜金色）+ 右侧标题

**板块6 - 联系CTA**
- 藏红背景
- "与我们合作"标题
- 联系方式（电话/地址/邮箱）
- 双按钮："联系我们" + "在线购买"

### 页面2：关于我们（About）

- **Banner**：深色渐变 + 顶部藏红金渐变装饰线 + 页面标题
- **公司简介**：左文（简介文字）+ 右色卡（藏红背景，核心业务列表）
- **企业使命与愿景**：三列布局
  - 使命（藏红顶线）：让高原珍品走出大山
  - 愿景（古铜金顶线）：成为甘南州领先的农产品电商服务平台
  - 价值观（草原绿顶线）：品质为本·诚信经营·文化传承·助农致富

### 页面3：产品展示（Products）

- **Banner**：绿色系渐变 + 页面标题
- **分类筛选**：8个胶囊标签（全部/食用菌/药材/畜产/野菜/粮油/蜂产品/工艺品）
- **产品网格**：3列卡片，每张卡片包含：
  - 产品图片区
  - 分类标签（彩色小标签）
  - 产品名称 + 描述
  - 底部分隔线：左"查看详情→" + 右产地信息
- **产品详情页**（子页面）：
  - 产品大图 + 名称 + 分类标签
  - 产品描述 + 产地信息 + 品质认证标签（绿色/有机/地理标志）
  - 购买渠道入口（微信二维码/淘宝店链接等外部链接）
  - 侧栏：相关产品推荐

### 页面4：文化故事（Culture）

- **Banner**：深色渐变 + 页面标题
- **主文化板块**：交替图文长卡片（左图右文/右图左文交替）
  - 土司历史文化：配红色系图片
  - 洮砚产业文化：配金色系图片
  - 每张卡片含：文化编号、标题、描述、"了解更多"链接
- **次要文化板块**：3个小卡片（1行3列）
  - 藏传佛教文化
  - 觉乃民俗文化
  - 自然生态文化

### 页面5：新闻资讯（News）

- **Banner**：大地棕渐变 + 页面标题
- **新闻列表**：图文横排卡片
  - 左侧缩略图（200px宽）
  - 右侧：分类标签 + 日期 + 标题 + 摘要
- **分类标签**：产业动态/品牌动态/活动通知

### 页面6：联系我们（Contact）

- **左右分栏布局**
- **左侧**（米白背景）：联系表单
  - 姓名、电话、邮箱、留言内容
  - 提交按钮（藏红）
- **右侧**（深色背景）：
  - 联系方式（地址/电话/邮箱）
  - 线上购买渠道（微信二维码/淘宝店铺/惠农商城）

## 通用组件

### 导航栏
- 左侧 Logo（"藏宝宝"品牌文字）
- 右侧菜单项：首页、关于我们、产品展示、文化故事、新闻资讯、联系我们
- 移动端：汉堡菜单 + 全屏侧边栏
- 滚动时添加白色背景和阴影

### Footer
- 深色背景（#1A1A2E）
- 顶部藏族纹样装饰线
- 四列布局：
  1. 品牌信息（Logo + 简介文字）
  2. 快速导航（页面链接）
  3. 产品分类（8大品类链接）
  4. 联系方式 + 社交媒体图标
- 底部版权信息

### 装饰元素
- 页面顶部渐变装饰线：藏红→古铜金→藏红（4px高）
- 分隔区域使用藏族纹样SVG图案（白色半透明）
- 卡片圆角统一 8px
- 按钮圆角统一 4px

## SEO 与性能

- 每个页面独立 meta title/description
- Open Graph 和 Twitter Card 标签
- 结构化数据（JSON-LD）：Organization、LocalBusiness
- 图片懒加载 + WebP/AVIF 自动转换
- Cloudflare Pages 全球 CDN 缓存
- Lighthouse 性能目标：Performance > 95、Accessibility > 90、SEO > 95

## 联系表单方案

静态站点使用 [Formspree](https://formspree.io) 或 [Cloudflare Workers](https://workers.cloudflare.com) 处理表单提交，无需自建后端。
